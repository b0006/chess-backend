# Chess (backend)

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running via Docker

```bash
# development
$ npm run docker:dev
```

```bash
# production
$ npm run docker:prod
```
